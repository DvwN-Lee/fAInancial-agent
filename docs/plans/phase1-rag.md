## 시작 조건 (트리거)

Phase 0 완료 후, 현재 Agent의 한계를 E2E 테스트로 확인하였다.

| 테스트 | 질문 | 결과 | 문제 |
|--------|------|------|------|
| 1 | 삼성전자 반도체 사업부 2024년 실적/전망 | "조회할 수 없습니다" | 사업부별 데이터 접근 수단 없음 |
| 2 | 삼성전자 최근 공시 리스크 요인 | 1238건 검색 목록만 반환 | 공시 문서 본문을 읽지 못함 |
| 3 | 삼성전자 AI 반도체 전략 | 1238건 검색 목록만 반환 | 공시 문서 본문을 읽지 못함 |

**공통 원인:** 현재 Agent는 공시 목록 검색(`dart_search`)과 재무제표 숫자 조회(`dart_financials`)만 가능하다. 공시 문서 본문에 접근하는 Tool이 없어 정성적 분석 질문에 답할 수 없다.
